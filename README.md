# Fireman

해당 프로그램은 개발 진행중입니다.

## 프로젝트 목적

Snowman의 동작 원리를 분석해, 디컴파일러를 작성한다.

## 코드 작성

### 코드 서식

#### 함수 주석

- \#\#\# Arguments
- \#\#\# Returns
- \#\#\# Note
- \#\#\# Todo

### 소스코드 내부 순서

#### 모듈 정의 파일 (mod.rs)

- 서브모듈에 대한 주석
- 서브모듈의 mod
- 서브모듈의 use
- 선언된 types

위를 반복한다.

타입이나 함수의 선언이 필요할 경우, 서브모듈을 만들어 pub use를 사용한다.

#### 소스코드 파일

- use문 정의
- 소스코드
- 구현 (구현이 작을 경우)
- 구현에 대한 서브모듈의 mod (구현이 많을 경우)
